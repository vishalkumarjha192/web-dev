# JavaScript DOM Manipulation Practice

This project contains basic examples of **HTML and JavaScript DOM
manipulation**.\
It helps beginners understand how JavaScript interacts with HTML
elements.

------------------------------------------------------------------------

## Topics Covered

### 1. Selecting Elements

-   getElementById()
-   getElementsByClassName()
-   querySelector()
-   querySelectorAll()

### 2. Text and Content Access

-   .innerText
-   .innerHTML
-   .textContent

### 3. Attribute Manipulation

-   .setAttribute()
-   .getAttribute()
-   .removeAttribute()

### 4. Dynamic DOM Manipulation

-   createElement()
-   append()
-   prepend()
-   appendChild()
-   remove()
-   removeChild()

### 5. Style Updates

-   .style
-   classList.add()
-   classList.remove()
-   classList.toggle()
